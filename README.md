# Ángela - Te Amo, Eres mi Universo

Un proyecto interactivo y romántico con animaciones en canvas.

## Características

✨ **Animaciones visuales:**
- Universo con estrellas parpadeantes
- Estrellas fugaces atravesando el cielo
- Corazones flotantes y giratorios
- Planeta rotatorio (el nombre "Ángela")
- Texto girando en órbita alrededor del título
- Mensajes románticos flotantes

💕 **Interactividad:**
- Clic para crear explosiones de corazones
- Movimiento del mouse genera corazones
- Control de música con botón

🎵 **Música:**
Agrega tu archivo de música como `music.mp3` en la raíz del repositorio.

## Instalación

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. (Opcional) Agrega `music.mp3` para la música

## Uso

- Mueve el mouse por la pantalla
- Haz clic para crear corazones
- Usa el botón de música (🎵) para activar/desactivar
